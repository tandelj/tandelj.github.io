Deep Convolutional Autoencoder for Image Denoising

This project uses a very interesting and effective application of deep learning, specifically Deep Convolutional Autoencoder. It is an implement an unsupervised deep convolutional autoencoder for image de-noising, converting noisy digits images from the MNIST dataset to clear digits images.

Visualize by AR

Automated Parking System

The goal was to Identify available parking spaces at bus depots and within the city. Designed and developed an application that allows user to locate and book parking at different places around the city. Developed a QR Code based system to verify the parking reservations and demonstrated its working with hardware prototype to different industry professionals. Provided solutions to real-time problems given by professionals. Built the prototype using Arduino and other hardware components. Real-time Database was built using Firebase.

Secure Voting

Implemented the Secure Election Protocol with two central facilities. The implementation provides a secure way for people to vote electronically, which eliminates the hassle of physically being present at designated election locations. Also, implemented RSA security algorithm for encryption and decryption. The communication between the users and central facilities is implemented using Java Socket programming using Client Server Architecture. It uses MySQL for the database through JDBC API connectivity.
